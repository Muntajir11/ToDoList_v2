# Todo List Website
Welcome to the Todo List Website! This platform is designed to help you organize your tasks efficiently and stay on top of your commitments. Whether you're managing personal projects, work tasks, or collaborative efforts, this website provides the tools you need to keep track of your todo lists effectively.

## Technologies Used
- **Database**: PostgreSQL is used as the database management system.
- **Backend Framework**: Node.js and Express.js provide the backend infrastructure for the website.
- **Frontend Templating**: EJS (Embedded JavaScript) is used for server-side rendering of dynamic content.
- **Styling**: CSS is utilized for styling the user interface.
- **Deployment**: The website can be deployed on various hosting platforms like Heroku or AWS.

## Features
- **Todo List Creation**: Create multiple to-do lists to categorize your tasks based on different projects or priorities.
- **Task Management**: Add, edit, mark as complete, and delete tasks within your to-do lists.
- **Persistent Storage**: Your to-do lists and tasks are stored securely in a PostgreSQL database, ensuring data persistence.
- **Responsive Design**: Enjoy a seamless user experience across various devices and screen sizes.

## Installation
To run this website locally, follow these steps:

1. Clone the repository:

   ```
   git clone https://github.com/Muntajir11/ToDoList_v2.git
   ```

2. Navigate to the project directory:

   ```
   cd <project-directory>
   ```

3. Install dependencies:

   ```
   npm install
   ```

4. Set up the PostgreSQL database with appropriate configurations.

5. Create a `.env` file in the root directory and define the following environment variables:

   ```
   DB_HOST=<database-host>
   DB_PORT=<database-port>
   DB_USER=<database-username>
   DB_PASSWORD=<database-password>
   DB_NAME=<database-name>
   SESSION_SECRET=<session-secret>
   ```

6. Run the application:

   ```
   npm start
   ```

7. Access the website at `http://localhost:3000` in your web browser.
